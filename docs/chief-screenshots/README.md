# Chief 5.0 reference screenshots

Place Chief 5.0 WhatsApp / screen captures in this folder for UI matching.

Design tokens used in the app (see `resources/css/app.css`):

- Active document tab: orange top border (`--chief-orange`)
- Action bar: dark blue / green (`--chief-action`, `--chief-action-green`)
- Selected grid row: bright blue (`--chief-selected`)
- Dense desktop grids, Favorite Lists left rail, status bar User | Site | Company
